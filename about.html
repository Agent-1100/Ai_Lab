About the Lab

A flexible structure for a modern AI research group.

This version separates the site into multiple HTML pages while keeping shared branding, shared styles, and shared navigation across the whole website.

Suggested sections to add next

  • Research themes for NLP, vision, robotics, theory, or domain AI areas.
  • Projects pages with demos, GitHub links, datasets, and benchmarks.
  • Join the lab page with application information and open positions.
  • Events pages for reading groups, seminars, workshops, and deadlines.
  • Media kit with logos, team photos, and citation details.

Why the multi-page structure helps

  • Each member can now have a real standalone profile URL.
  • GitHub Pages works naturally with this file structure.
  • Shared CSS and JavaScript reduce repetition and simplify edits.
  • It will be easier to scale into a larger lab website later.
  • You can still convert this into a CMS or framework project later.
